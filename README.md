# odin-recipes
Odin-recipes is a project who's purpose is to create recipes for my favorite meal. I have created three recipes. They contain the title of my favorite dish. I have also listed the ingredients for cooking the meal. Finally I have explained the steps followed to prepare the meal. 

Through this project I have learnt:
1. How to create and clone a repository. 
2. How to create the homepage(index.html)
3. How to create pages and linking them to the index page.
4. I have also learnt how to add images to a page, how to create lists, and heading elements.
